# 🌍 Global Electronics Retailer Analysis & Dashboard

## 📊 Project Overview
This project analyzes a global electronics retailer dataset to explore sales performance, customer behavior, and product insights.  
The objective is to transform raw data into structured analysis and build a dashboard that supports data-driven decision-making.

---

## 🧹 Data Cleaning
Data preparation was performed using Excel:
- Removed duplicate records  
- Handled missing and inconsistent values  
- Standardized date and numeric formats  
- Structured the dataset for analysis  

---

## 🔄 Data Transformation
Data transformation steps included:
- Organizing and reshaping raw data  
- Creating calculated columns  
- Preparing structured data for analysis and visualization  
- Utilizing Power Query for data transformation and cleaning  

---

## 📈 Data Analysis
The following analyses were performed:

- Sales performance analysis across time  
- Product performance evaluation  
- Geographic analysis by region and country  
- Customer segmentation and behavior analysis  

---

## 📊 Dashboard
The dashboard is structured into the following sections:

- 📈 Sales Performance  
- 📦 Product Performance  
- 🌍 Geographic Performance  
- 👥 Customer Segmentation  

Each section provides focused insights to support business understanding and decision-making.

---

## 🛠 Tools Used
- Microsoft Excel  
- Power Query  
- Power BI
